# **App Name**: Brisa's Portfolio

## Core Features:

- Navigation: Intuitive navigation bar linking to each section.
- About Section: Display Brisa's bio, a headshot, and inspiration sources.
- Project Showcase with Transitions: Highlight personal projects (Mi yo vegetal) with detailed descriptions, images, and animated entry transitions.
- Branding Samples with Dynamic Content: Showcase branding projects, transitioning between 'Impulso Azul' and 'Zelda Tattoo Supplies' sections as the user scrolls. Include relevant images and descriptions.
- Autoscrolling Illustration Gallery: Implement an autoscrolling gallery of illustrations, ensuring smooth and visually appealing transitions.
- Contact Section with Links: Provide contact information, including links to Instagram, WhatsApp, and email with hover animations.

## Style Guidelines:

- Primary color: Lavender (#E6D7F2), reminiscent of the artistic mood and the lilac text color shown in the reference image.
- Background color: Dark gray (#222222), for good contrast with other elements in a dark color scheme.
- Accent color: Pale rose (#F2D7EE), for secondary elements like icons and highlights.
- Body and headline font: 'Literata', a serif font that communicates a vintage and slightly formal feel which matches well with Brisa's persona and portfolio content.
- Simple, outlined icons for the navigation and contact sections.
- A clean, single-page layout that focuses on visual content.
- Smooth, subtle animations for transitions and hover effects.
